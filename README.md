# Flash — Study Flashcards

Interactive flashcard app for studying CS topics. Select topics, choose how many cards, flip through questions with code snippets, and track your progress.

## Topics

- Algorithmic Complexity / Big-O
- Arrays
- Linked Lists
- Stacks
- Queues
- Hash Tables
- Binary Search
- Bitwise Operations
- Trees (BST, Heaps, Traversals)
- Sorting Algorithms
- Graphs (BFS, DFS, Dijkstra)
- Recursion & Backtracking
- Dynamic Programming
- Design Patterns
- Caches
- Processes & Threads
- Networking
- Tries
- System Design
- String Manipulation

*Code snippets are written in Python.*

## Stack

- React 19
- TypeScript
- Tailwind CSS 4
- Vite

## Running

```bash
cd flash
npm install
npm run dev
```

## Features

- Pick topics and number of cards
- Flip cards with click/tap or keyboard (Space)
- Mark cards as "Got It" or "Still Learning"
- Progress bar and score summary
- Per-topic breakdown with progress bars
- Retry missed cards
- Keyboard shortcuts (Space to flip, arrow keys to navigate)
- Mobile responsive
